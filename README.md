# 🌍 World GDP Analysis

**World GDP Analysis** is a Python-based data analysis project that explores global GDP trends from 1960 to 2025. This project demonstrates real-world data cleaning, visualization, and insights generation, making it perfect for a **portfolio or resume**.

---

## 🚀 Project Overview

The global economy is constantly evolving, and understanding GDP trends can provide valuable insights into economic growth, global shifts, and country-level performance. This project:

- Loads and cleans GDP data from World Bank sources.
- Visualizes GDP trends for top countries over time.
- Highlights historical economic patterns and growth comparisons.
- Uses Python libraries like **Pandas**, **Matplotlib**, and **Seaborn**.
